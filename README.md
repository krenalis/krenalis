### Key Features

* A **(CDP) Customer Data Platform** for efficient data management
* Data is stored **on your own data warehouse**
* Data pipelines with an **advanced type system**
* **Ingest**, **activate** and **export** **users and events**
* **User unification**
* **Segment** and **RudderStack** compatibility

### Integrations

Meergo can interface and exchange data with services such as:

* **Services**. HubSport, Mailchimp, Segment, etc...
* **Databases**. PostgreSQL, MySQL, etc...
* **Website, mobile apps**. Using one of the available **Meergo SDK**.
* CSV, Excel, Parquet, JSON files, etc... on your storage (S3, HTTP, FTP...)
 
<b>See the [full list of integrations](http://localhost:8080/connectors/) available in Meergo.</b>

## Getting Started


If you have Docker and Git installed, try Meergo locally by simply running:

```
git clone https://github.com/meergo/meergo.git && cd meergo && docker compose up
```

Otherwise, you can checkout the documentation page about [how to install Meergo](http://localhost:8080/installation).
