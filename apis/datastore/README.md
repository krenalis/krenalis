# Datastore

## Type (With Constraints) Support In Data Warehouses

**IMPORTANT**. Please note that this table is under construction / revision, so verify the information before relying on them.

This table is based on the following principles:

* a type $T$ (with any constraints) is considered supported by a data warehouse if the data warehouse provides a type whose set of values ​​exactly matches the set of values ​​of $T$
* if a certain type $T$ (with any constraints) is not supported by any data warehouse, then it must be prohibited in the user schema
