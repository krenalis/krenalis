## How to test Python functions

First, install [pytest](https://docs.pytest.org/en/7.4.x/getting-started.html).

Then, within this directory, just run:

```
pytest -vs
```

Run `pytest --help` for a command-line usage of Pytest.

Also see [the online documentation of Pytest](https://docs.pytest.org/en/7.4.x/) for instructions on how Pytest works and how to add new tests.
