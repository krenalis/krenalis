## Environment variables necessary for testing

* `MEERGO_TEST_GOOGLE_ANALYTICS_MEASUREMENT_ID`. Measurement ID, see: https://support.google.com/analytics/answer/12270356?hl=en.
* `MEERGO_TEST_GOOGLE_ANALYTICS_API_SECRET`. API secret, passed as `api_secret`. See: https://developers.google.com/analytics/devguides/collection/protocol/ga4/reference?client_type=gtag.
* `MEERGO_TEST_GOOGLE_ANALYTICS_COLLECTION_ENDPOINT`. Collection endpoint; can be "Global" or "EU".
