
## What can you do with this?

This connector lets you send events from the JavaScript on your website or web application to Meergo. The events can be streamed in real time to your data warehouse and to SaaS applications. Users data from events can be imported into the data warehouse to be consolidated into unified user profiles within Meergo.

## What does it require?

* Integration of the Meergo JavaScript SDK into your website or application.
* Basic knowledge of JavaScript or TypeScript and the ability to write code.
