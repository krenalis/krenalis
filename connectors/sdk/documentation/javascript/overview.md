
## What can you do with this?

This connector lets you send events from the JavaScript on your website or web application directly to Meergo. Both the events and their associated user data can be streamed in real time to your data warehouse. User data can also be consolidated into unified user profiles within Meergo.

## What does it require?

* Integration of the Meergo JavaScript SDK into your website or application.
* Basic knowledge of JavaScript or TypeScript and the ability to write code.
