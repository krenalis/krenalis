
## What can you do with this?

This connector lets you send events from the JavaScript on your website or web application to Krenalis. The events can be streamed in real time to your data warehouse and to SaaS applications. Users data from events can be imported into the data warehouse to be consolidated into unified user profiles within Krenalis.

## What does it require?

* Integration of the Krenalis JavaScript SDK into your website or application.
* Basic knowledge of JavaScript or TypeScript and the ability to write code.
