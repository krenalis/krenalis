
## What can you do with this?

This connector lets you send events from your Python application to Meergo. The events can be streamed in real time to your data warehouse and to SaaS applications. Users data from events can be imported into the data warehouse to be consolidated into unified user profiles within Meergo.

## What does it require?

* Integration of the Meergo Python SDK into your application.
* Basic knowledge of Python and the ability to write code.
