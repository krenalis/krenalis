
## What can you do with this?

This connector lets you send events from your node.js application directly to Meergo. Both the events and their associated user data can be streamed in real time to your data warehouse. User data can also be consolidated into unified user profiles within Meergo.

## What does it require?

* Integration of the Meergo Node.js SDK into your application.
* Basic knowledge of JavaScript or TypeScript and the ability to write code.
