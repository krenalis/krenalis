
S3 (Simple Storage Service) is a scalable object storage service provided by Amazon Web Services (AWS) that allows you to store and retrieve any amount of data from anywhere on the web.

## What can you do with this?

This connector exports unified users (i.e., users consolidated through identity resolution) from your data warehouse in various file formats, such as CSV or Excel, and sends them directly to S3 buckets

## What does it require?

* An S3 account with a bucket to write files to.
