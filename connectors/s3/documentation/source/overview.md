
S3 (Simple Storage Service) is a scalable object storage service provided by Amazon Web Services (AWS) that allows you to store and retrieve any amount of data from anywhere on the web.

## What can you do with this?

This connector lets you import user data from files stored in S3 buckets into your data warehouse. You can unify this data as users within Meergo by importing files in various formats, such as CSV, Excel, or others.

## What does it require?

* An S3 account with a bucket to read files from.
