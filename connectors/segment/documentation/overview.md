
Segment is a leading customer data platform that collects, unifies, and integrates customer data to enhance analytics and personalization.

## What can you do with this?

Using this connector, which interfaces with Segment, you can integrate customer event data from Segment into Meergo in real time. This allows you to:

* Stream customer events in real time to your data warehouse and to SaaS applications.
* Import user data into your data warehouse and unify it into Meergo's user profiles.

## What does it require?

* Any [Segment](https://segment.com/) account.
