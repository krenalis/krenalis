
HubSpot is a cloud application that offers tools for customer relationship management (CRM), marketing, and sales. It helps businesses organize contacts, automate marketing campaigns, and improve customer support.

## What can you do with this?

Using this connector you can read contacts from HubSpot, import them into your data warehouse, and unify them as users in Krenalis.

## What does it require?

* Any [HubSpot](https://www.hubspot.com/) account.
* A [HubSpot developer account](https://developers.hubspot.com/).
* The ability to set environment variables when starting Krenalis.
