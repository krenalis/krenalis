
UISample is a connector for testing and previewing UI controls. 

## What can you do with this?

This connector lets you explore and interact with all the available UI controls that can be used to build user interfaces for other connectors.

Its sole purpose is to serve as a testing ground or showcase, allowing developers and integrators to see how each control behaves, what kind of input it accepts, and how it can be configured.
You can use this connector to prototype interface layouts, validate settings forms, or simply to get familiar with the UI components supported by Krenalis. 

## What does it require?

Nothing.
