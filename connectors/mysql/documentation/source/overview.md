
MySQL is an open-source relational database management system. It's popular for web applications due to its scalability, security, and performance.

## What can you do with this?

This connector lets you read users from a MySQL database, store them in your data warehouse, and unify them as users within Meergo.

## What does it require?

* A MySQL database to read the users from.
