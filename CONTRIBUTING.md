## How to report a bug

1. Go to the [issues list](https://github.com/meergo/meergo/issues).
2. Check that there is not already an issue for the problem you are experiencing. If so, instead of creating a duplicate issue that would be removed, you can add your feedback to the existing one.
3. If you do not find any issues for the problem you are experiencing, you can open a new one through the [appropriate template](https://github.com/meergo/meergo/issues/new?template=new-issue.md).
